删除变更是指通过删除操作来修改表数据的 `ALTER` 查询，最典型的就是 `ALTER TABLE DELETE` 等语句。执行此类查询会为数据分片生成新的变更版本。这意味着，此类语句会对所有在变更之前插入的数据分片进行整体重写，从而产生大量写入请求。

:::info
对于删除场景，可以使用 [ReplacingMergeTree](/guides/replacing-merge-tree) 或 [CollapsingMergeTree](/engines/table-engines/mergetree-family/collapsingmergetree) 等专用表引擎替代默认的 MergeTree 表引擎，从而避免产生如此大量的写入请求。
:::
