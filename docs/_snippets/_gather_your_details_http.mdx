要通过 HTTP(S) 连接到 ClickHouse，你需要准备以下信息：

| Parameter(s)              | Description                             |
| ------------------------- | --------------------------------------- |
| `HOST` and `PORT`         | 使用 TLS 时通常为 8443，未使用 TLS 时通常为 8123。     |
| `DATABASE NAME`           | 系统默认提供一个名为 `default` 的数据库；填写你要连接的数据库名称。 |
| `USERNAME` and `PASSWORD` | 系统默认提供用户名 `default`；请使用适合你场景的用户名。       |

你可以在 ClickHouse Cloud 控制台中查看 ClickHouse Cloud 服务的详细信息。
选择一个服务并点击 **Connect**：

<Image img={cloud_connect_button} size="md" alt="ClickHouse Cloud 服务 Connect 按钮" border />

选择 **HTTPS**。连接详情会显示在示例 `curl` 命令中。

<Image img={connection_details_https} size="md" alt="ClickHouse Cloud HTTPS 连接详情" border />

如果你使用的是自管 ClickHouse，则连接详情由你的 ClickHouse 管理员进行配置。
