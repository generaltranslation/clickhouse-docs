* `config.d` 目录用于存放 ClickHouse 服务器配置片段，这些配置会与每次安装 ClickHouse 时提供的默认配置文件 `config.xml` 合并，可用于为各个 ClickHouse 节点定义自定义配置。
* `users.d` 目录用于存放用户配置片段，这些配置会与每次安装 ClickHouse 时提供的默认用户配置文件 `users.xml` 合并，可用于定义用户级的自定义配置。

:::tip 自定义配置目录
在编写自定义配置时，最佳实践是使用 `config.d` 和 `users.d` 目录，而不是直接修改 `/etc/clickhouse-server/config.xml` 和 `/etc/clickhouse-server/users.xml` 中的默认配置。

这行

```xml
<clickhouse replace="true">
```

确保在 `config.d` 和 `users.d` 目录中定义的配置节会覆盖默认 `config.xml` 和 `users.xml` 文件中定义的配置节。
:::
