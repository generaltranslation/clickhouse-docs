对于每个服务器，需要指定以下参数：

| Parameter     | Description                                                                                                                                                                                                                                                                                                                                          | Default Value |
|---------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------|
| `host`        | 远程服务器的地址。可以是域名或 IPv4 / IPv6 地址。如果指定的是域名，服务器在启动时会发起 DNS 请求，并在运行期间缓存解析结果。如果 DNS 请求失败，服务器将无法启动。更改 DNS 记录后，需要重启服务器。 | -            |
| `port`        | 用于消息传递的 TCP 端口（配置中的 `tcp_port`，通常为 9000）。不要与 `http_port` 混淆。                                                                                                                                                                                                                           | -            |
